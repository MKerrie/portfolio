# Portfolio - Kerim Orgu

Mijn persoonlijke portfolio website als Full-Stack Developer.

## Live Demo

[Bekijk de website](https://mkerrie.com)

## Features

- Modern, responsive design
- Dark/Light mode toggle
- Animated particles achtergrond
- Smooth scroll animaties
- Contact formulier met EmailJS
- Blog sectie
- Project showcase
- Skills overzicht met flip cards

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Styling:** Tailwind CSS
- **Animaties:** Custom CSS & JavaScript
- **Icons:** Font Awesome, Bootstrap Icons
- **Contact:** EmailJS

## Secties

- Home - Hero sectie met animaties
- Over Mij - Persoonlijke introductie
- Skills - Technische vaardigheden
- Projecten - Portfolio projecten
- Blog - Technische artikelen
- Contact - Contact formulier

## Projecten

1. **MediTrack** - Medicatie tracking app
2. **DierenAsiel** - Dierenasiel management systeem
3. **FitForge** - Fitness tracking app
4. **Piccobello** - Foodtruck website
